# Pure CSS 3D Grid Project

This is a beginner-friendly front-end project created after completing
HTML and CSS fundamentals.  
The purpose of this project is to practice layout design, hover effects,
and modern UI styling using only CSS.

##  Features
- Pure HTML & CSS (No JavaScript)
- 3D hover animation using CSS transforms
- Glassmorphism UI effect
- Clean and modern layout
- Beginner-friendly project structure

## Technologies Used
- HTML5
- CSS3 (Flexbox, Transforms, Transitions)

##  Project Purpose
This project was built to strengthen CSS fundamentals and understand:
- Box model
- Flexbox layout
- Hover effects
- CSS transitions and transforms
- UI design basics

## Learning Outcome
By building this project, I learned how to:
- Create responsive layouts using Flexbox
- Apply 3D effects with CSS
- Design visually appealing UI components
- Write clean and structured CSS
